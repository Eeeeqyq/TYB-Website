# TYB Website

Frontend-only TYB site refresh.
Same content, cleaner Blackstone-ish look.

## Stack

- HTML
- CSS
- Vanilla JS

## Files

- `index.html` - all page content
- `styles.css` - all styling / layout
- `script.js` - nav toggle + reveal animations

## Run

Just open `index.html` in browser.

If you want local server:

```bash
python3 -m http.server 8000
```

Then go to `http://localhost:8000`.

## Edit Guide

- Change text/content -> `index.html`
- Change look/spacing/colors -> `styles.css`
- Change interactions -> `script.js`

## Notes

- Mobile responsive.
- No build step.
- No framework.
