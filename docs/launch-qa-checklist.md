# Launch QA Checklist

## Pre-Launch
- [ ] Homepage loads over HTTPS.
- [ ] Title and meta description appear correctly in page source.
- [ ] Canonical tag exists and points to preferred homepage URL.
- [ ] Open Graph and Twitter tags present (`og:title`, `og:description`, `og:image`, `twitter:card`).
- [ ] Organization JSON-LD script present and valid.
- [ ] `robots.txt` reachable.
- [ ] `sitemap.xml` reachable and valid XML.
- [ ] No broken local assets (logo, CSS, JS, OG image).
- [ ] Mobile layout and navigation work on phone-sized viewport.
- [ ] Contact CTA (`mailto:`) opens correctly.

## Day-1 Post-Launch
- [ ] Google Search Console property verified.
- [ ] Sitemap submitted in Search Console.
- [ ] Homepage indexing requested.
- [ ] Non-canonical host redirects to canonical host (301/308).

## First 2-4 Weeks
- [ ] Homepage is indexed.
- [ ] Brand query begins appearing in Google results.
- [ ] Search Console has no critical crawl/index errors.
- [ ] Track weekly inquiry signals (email clicks/leads).
